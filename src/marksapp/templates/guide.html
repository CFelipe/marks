Importing

You can import your bookmarks from Chrome, Firefox, Safari, Opera, Internet Explorer, as long as they're in the Netscape bookmark format. Your directories will be converted to tags automatically. If you have a bookmark inside the folders "Photography -> Music" they'll get the tags "Photography" and "Music". If you have a bookmark inside the folders "Music -> Photography", they'll get the same tags as the other bookmark.

Adding bookmarks

URL

That's the link to your bookmarked site. If you copy and paste it, the title will be fetched from the page automatically and added to the Name field as long as it is blank.

Name

The name of your bookmark.

Tags

Comma-separated tags for your bookmark.

Private marks

If you add the "private" tag to a bookmark, it'll become hidden to the public.

Unlisted marks

If you prefix a tag with ".", it won't show up in the public tag view, but the bookmarks will be visible if you share the tag's URL.

Descriptions

When you add a bookmark, you can write a description for it using Markdown syntax.

Viewing bookmarks

You can sort bookmark listings by name or by date (most recent first). The description for a bookmark is hidden by default and can be seen by clicking on info, or simply clicking expand all.
